App Overview

Built on Node.js, this app uses the Google Analytics API to fetch your data and converts visits into a color somewhere between red and green and powers an LED. Connecting an Arduino to Google Analytics is not as hard as it may seem thanks to the wonderful noduino library created by semu.

What you will need

Hardware
  • Arduino Board
  • Wires
  • Raspberry Pi
  • PowerSSR
Software
  • Node.js (I'm using v0.8.11)
  • Arduino .ino
